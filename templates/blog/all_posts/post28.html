문자열 뒤집는 함수를 작성하라. 입력값은 문자 배열이며, 리턴 없이 리스트 내부를 조작하라

투 포인터를 이용한 스왑

def reverseString(s: list[str]) -> None:
    left, right = 0, len(s) -1
    while left < right:
        s[left], s[right] = s[right], s[left]
        left += 1
        right -= 1
  • 두 개의 포인터를 이용해 범위를 조정해가며 풀이하는 방식
  • '리턴없이 리스트 내부를 직접 조작하라'는 제약 사항이 있으므로 위와 같이 s 내부를 스왑하는 형태로 풀이하면 됨

파이썬다운 방식

def reverseString(s: list[str]) -> None:
	s.reverse()
  • 파이썬의 기본 기능 이용

슬라이싱 이용

def reverseString(s: str) -> str:
    s = s[::-1]
  • 하지만 공간복잡도 O(1) 제약이 걸린 leetcode는 [::-1]이 먹히지 않으므로 s[:]=s[::-1]을 사용하면 됨
  • 코딩 테스트가 이루어지는 플랫폼별로 제한이 있으므로, 플랫폼 특징에 대한 충분한 숙지가 필요함
